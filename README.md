# Eduk8me Clocks

__([Full list of silly diversions from Eduk8.me Labs](https://labs.eduk8.me))__

![Screenshots of clocks](https://eduk8me.github.io/clocks/assets/eduk8meclocks.png)

There are three clocks available:

- A [plain old clock](https://eduk8me.github.io/clocks/clock.html)
- An [iOS 9-compatible plain old clock](https://eduk8me.github.io/clocks/clock-ios9.html)
- A [countdown clock](https://eduk8me.github.io/clocks/countdown.html)

These clocks were written with the help of ChatGPT. The countdown clock was written as an easy way to bring in a countdown clock into [OBS](https://obsproject.com/) as a browser source. 

When using it in OBS, a good starting size for the browser source is 1920x300. The size can be adjusted as necessary to fit.

# Usage

Mousing over or tapping the bottom right corner will show a settings gear to modify aspects of each clock. The settings are stored in the url, so once you have the clock or countdown set up the way you want, save a bookmark to the page.
